# 🌱 EcoPoints

Projeto web desenvolvido para incentivar práticas sustentáveis através de um sistema de pontuação gamificado.
link do video: https://youtu.be/C6rd5JN34W0

## 🚀 Sobre o projeto


O **EcoPoints** é um projeto com foco na sustentabilidade, onde o usuário realiza ações do dia a dia que contribuem para o meio ambiente e, em troca, ganha pontos. Essas ações incentivam hábitos mais conscientes, como reciclar, economizar recursos e utilizar meios de transporte sustentáveis.

O sistema conta com um ranking, onde os pontos acumulados permitem comparar o desempenho entre usuários, trazendo um elemento de gamificação para tornar a experiência mais envolvente e motivadora.

A ideia principal é unir tecnologia e sustentabilidade, incentivando pequenas atitudes que fazem diferença no mundo 🌱

## 🎮 Funcionalidades

* ✅ Seleção de ações sustentáveis
* ➕ Sistema de pontuação dinâmica
* 🏆 Ranking de usuários
* 🔐 Tela de login
* 📱 Layout responsivo (mobile, tablet e desktop)

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
