Sobre o EcoPoints

Conectando tecnologia e sustentabilidade para transformar hábitos 🌱

Contexto do Projeto

O EcoPoints surge diante da necessidade de incentivar práticas sustentáveis no dia a dia das pessoas. Atualmente, muitos problemas ambientais, como o excesso de resíduos plásticos e a emissão de gases poluentes, estão diretamente ligados aos hábitos da população.

Apesar da crescente conscientização, ainda falta engajamento e incentivo para que as pessoas adotem atitudes mais sustentáveis de forma contínua. Pensando nisso, o projeto busca unir tecnologia e sustentabilidade para gerar impacto positivo.

Solução Proposta

O EcoPoints propõe uma plataforma que incentiva ações sustentáveis por meio de um sistema de pontuação (gamificação). Os usuários recebem pontos ao realizar atividades ecológicas, como utilizar ecobags, andar de bicicleta e reciclar.

Além disso, a plataforma conta com um feed social onde os usuários podem compartilhar suas ações, interagir com outros membros e trocar dicas sustentáveis, criando uma comunidade engajada.

O sistema também inclui ranking de usuários e recompensas, incentivando a participação contínua.

Tecnologias Utilizadas

  • Frontend: HTML, CSS e JavaScript
  • Backend (planejado): Node.js ou Python (Django)
  • Banco de Dados: PostgreSQL ou MongoDB
  • Controle de versão: Git e GitHub

Roadmap do Projeto

  • ✔️ Criação da interface inicial (HTML e CSS)
  • 🔄 Implementação do sistema de cadastro e login
  • 🔄 Desenvolvimento do sistema de pontuação
  • 🔄 Criação do feed social
  • 🔄 Implementação do ranking de usuários
  • 🔄 Desenvolvimento do painel administrativo
  • 🚀 Lançamento da versão inicial (MVP)
